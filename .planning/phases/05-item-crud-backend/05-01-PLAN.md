Also add a `dedup` helper function:
```go
// dedup returns a new slice with duplicate strings removed, preserving order.
func dedup(ss []string) []string {
    seen := make(map[string]bool, len(ss))
    result := make([]string, 0, len(ss))
    for _, s := range ss {
        if !seen[s] {
            seen[s] = true
            result = append(result, s)
        }
    }
    return result
}
```

**store.go additions:**

Add the following methods to the Store struct. All methods take `context.Context` as first parameter for cancellation propagation.

**1. CreateItem** (per D-11, D-14, D-17, ITEM-02):
- Signature: `func (s *Store) CreateItem(ctx context.Context, containerID int64, name string, description *string, tags []string) (*ItemResponse, error)`
- Begin transaction
- Check container exists: `SELECT id FROM container WHERE id = ?` -- if no row, return `fmt.Errorf("container not found")` (caller maps to 404 per D-11)
- Insert item: `INSERT INTO item (container_id, name, description, created_at, updated_at) VALUES (?, ?, ?, datetime('now'), datetime('now'))`
- Get itemID via `result.LastInsertId()`
- For each tag (already lowercase-normalized and deduplicated by caller):
  - `INSERT OR IGNORE INTO tag (name, created_at, updated_at) VALUES (?, datetime('now'), datetime('now'))` (per D-14: auto-create)
  - `SELECT id FROM tag WHERE name = ?`
  - `INSERT INTO item_tag (item_id, tag_id) VALUES (?, ?)`
- Commit transaction
- Call `s.GetItem(ctx, itemID)` to return the full item with tags and container_label

**2. GetItem** (per D-01, D-02, D-04, D-21):
- Signature: `func (s *Store) GetItem(ctx context.Context, id int64) (*ItemResponse, error)`
- Query: `SELECT i.id, i.container_id, i.name, i.description, c.col, c.row, i.created_at, i.updated_at FROM item i JOIN container c ON c.id = i.container_id WHERE i.id = ?`
- If `sql.ErrNoRows`, return `nil, nil`
- Compute `ContainerLabel` via `labelFor(col, row)`
- Fetch tags: `SELECT t.name FROM tag t JOIN item_tag it ON it.tag_id = t.id WHERE it.item_id = ? ORDER BY t.name`
- If tags is nil, set to empty slice `[]string{}` (so JSON serializes as `[]` not `null`)
- Format CreatedAt/UpdatedAt as RFC3339 strings

**3. UpdateItem** (per D-13, D-18):
- Signature: `func (s *Store) UpdateItem(ctx context.Context, id int64, name string, description *string, containerID int64) (*ItemResponse, error)`
- Begin transaction
- Check item exists: `SELECT id FROM item WHERE id = ?` -- if no row, return `nil, nil`
- Check target container exists: `SELECT id FROM container WHERE id = ?` -- if no row, return error "container not found"
- Update: `UPDATE item SET name = ?, description = ?, container_id = ?, updated_at = datetime('now') WHERE id = ?`
- Commit
- Return `s.GetItem(ctx, id)`
- NOTE: per D-18, does NOT touch tags

**4. DeleteItem** (per ITEM-04):
- Signature: `func (s *Store) DeleteItem(ctx context.Context, id int64) error`
- Check item exists first: `SELECT id FROM item WHERE id = ?` -- if no row, return `fmt.Errorf("item not found")`
- `DELETE FROM item WHERE id = ?` -- CASCADE handles item_tag cleanup
- Return nil

**5. AddTagToItem** (per D-14, D-16):
- Signature: `func (s *Store) AddTagToItem(ctx context.Context, itemID int64, tagName string) (*ItemResponse, error)`
- Begin transaction
- Check item exists: `SELECT id FROM item WHERE id = ?`
- `INSERT OR IGNORE INTO tag (name, created_at, updated_at) VALUES (?, datetime('now'), datetime('now'))`
- `SELECT id FROM tag WHERE name = ?`
- `INSERT OR IGNORE INTO item_tag (item_id, tag_id) VALUES (?, ?)` (IGNORE handles already-linked case)
- Update item.updated_at: `UPDATE item SET updated_at = datetime('now') WHERE id = ?`
- Commit
- Return `s.GetItem(ctx, itemID)`

**6. RemoveTagFromItem** (per D-15, D-16):
- Signature: `func (s *Store) RemoveTagFromItem(ctx context.Context, itemID int64, tagName string) error`
- Get tag ID: `SELECT id FROM tag WHERE name = ?` -- if not found, return `fmt.Errorf("tag not found")`
- Delete junction: `DELETE FROM item_tag WHERE item_id = ? AND tag_id = ?`
- Check rows affected -- if 0, return `fmt.Errorf("tag not associated with item")`
- Update item.updated_at: `UPDATE item SET updated_at = datetime('now') WHERE id = ?`
- Per D-15: do NOT delete the tag from tag table (orphaned tags kept)

**7. ListItemsByContainer** (per D-20):
- Signature: `func (s *Store) ListItemsByContainer(ctx context.Context, containerID int64) (*ContainerWithItems, error)`
- Query container: `SELECT id, shelf_id, col, row, created_at, updated_at FROM container WHERE id = ?`
- If not found, return `nil, nil`
- Compute label via `labelFor(col, row)`
- Query items: `SELECT id FROM item WHERE container_id = ? ORDER BY name`
- For each item ID, call `s.GetItem(ctx, id)` to get full item with tags
- Return ContainerWithItems with items array (empty `[]ItemResponse{}` if no items)

**8. ListAllItems** (per D-22):
- Signature: `func (s *Store) ListAllItems(ctx context.Context) ([]ItemResponse, error)`
- Query all item IDs: `SELECT id FROM item ORDER BY name`
- For each, call `s.GetItem(ctx, id)`
- Return slice (empty `[]ItemResponse{}` if none)

**9. ListTags** (per D-19):
- Signature: `func (s *Store) ListTags(ctx context.Context, prefix string) ([]TagResponse, error)`
- If prefix is empty: `SELECT id, name, created_at, updated_at FROM tag ORDER BY name`
- If prefix non-empty: same query + `WHERE name LIKE ?` with arg `prefix + "%"`
- Return slice (empty `[]TagResponse{}` if none)
- Format timestamps as RFC3339

All methods must use `context.Context` parameter (pass to `ExecContext`, `QueryContext`, `QueryRowContext`).
All transaction methods use `defer tx.Rollback()` pattern -- Rollback is no-op after Commit.
