func writeJSON(w http.ResponseWriter, status int, v any) {
    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(status)
    json.NewEncoder(w).Encode(v)
}

func writeError(w http.ResponseWriter, status int, msg string) {

}

Validation function for CreateItemRequest (per D-06 through D-12):
- Trim name (`strings.TrimSpace`), reject empty -- error: "name is required"
- Reject name > 200 chars -- error: "name must be at most 200 characters"
- Reject len(Tags) == 0 -- error: "at least one tag is required" (per D-07)
- Reject len(Tags) > 20 -- error: "at most 20 tags allowed" (per D-08)
- For each tag: trim, lowercase (`strings.ToLower(strings.TrimSpace(t))`), reject empty after trim -- error: "tag must not be empty", reject > 50 chars -- error: "tag must be at most 50 characters" (per D-09)
- Deduplicate tags via `dedup()` (per D-10)
- If description non-nil: reject len > 1000 -- error: "description must be at most 1000 characters" (per D-12)
- Reject container_id <= 0 -- error: "container_id is required"

Validation for UpdateItemRequest: same rules for name and description, plus container_id > 0 required.

Validation for AddTagRequest: trim, lowercase, reject empty, reject > 50 chars.

**handlers.go -- Handler functions:**

All handlers are closures that capture `*Store`:

1. `handleCreateItem(store *Store) http.HandlerFunc` -- POST /api/items
   - Decode JSON body via `json.NewDecoder(r.Body).Decode(&req)`
   - Validate, return 400 on failure
   - Call `store.CreateItem(r.Context(), req.ContainerID, req.Name, req.Description, req.Tags)`
   - If error contains "container not found", return 404 (per D-11)
   - On success, `writeJSON(w, http.StatusCreated, item)`

2. `handleGetItem(store *Store) http.HandlerFunc` -- GET /api/items/{itemID}
   - Parse itemID: `strconv.ParseInt(chi.URLParam(r, "itemID"), 10, 64)`
   - If parse error, return 400 "invalid item ID"
   - Call `store.GetItem(r.Context(), id)`
   - If nil, return 404 "item not found"
   - `writeJSON(w, http.StatusOK, item)`

3. `handleUpdateItem(store *Store) http.HandlerFunc` -- PUT /api/items/{itemID}
   - Parse itemID from URL
   - Decode + validate UpdateItemRequest
   - Call `store.UpdateItem(r.Context(), id, req.Name, req.Description, req.ContainerID)`
   - If nil result, return 404 "item not found"
   - If error contains "container not found", return 404
   - `writeJSON(w, http.StatusOK, item)`

4. `handleDeleteItem(store *Store) http.HandlerFunc` -- DELETE /api/items/{itemID}
   - Parse itemID from URL
   - Call `store.DeleteItem(r.Context(), id)`
   - If error contains "item not found", return 404
   - On success, `w.WriteHeader(http.StatusNoContent)` (204, no body)

5. `handleAddTag(store *Store) http.HandlerFunc` -- POST /api/items/{itemID}/tags
   - Parse itemID from URL
   - Decode AddTagRequest, validate tag name (trim, lowercase, non-empty, max 50)
   - Call `store.AddTagToItem(r.Context(), id, req.Name)`
   - If error contains "item not found", return 404
   - `writeJSON(w, http.StatusOK, item)`

6. `handleRemoveTag(store *Store) http.HandlerFunc` -- DELETE /api/items/{itemID}/tags/{tagName}
   - Parse itemID and tagName from URL: `chi.URLParam(r, "tagName")`
   - Lowercase tagName: `strings.ToLower(tagName)`
   - Call `store.RemoveTagFromItem(r.Context(), id, tagName)`
   - If error contains "not found" or "not associated", return 404
   - On success, `w.WriteHeader(http.StatusNoContent)`

7. `handleListContainerItems(store *Store) http.HandlerFunc` -- GET /api/containers/{containerID}/items
   - Parse containerID from URL
   - Call `store.ListItemsByContainer(r.Context(), id)`
   - If nil, return 404 "container not found"
   - `writeJSON(w, http.StatusOK, result)`

8. `handleListItems(store *Store) http.HandlerFunc` -- GET /api/items
   - Call `store.ListAllItems(r.Context())`
   - `writeJSON(w, http.StatusOK, items)`

9. `handleListTags(store *Store) http.HandlerFunc` -- GET /api/tags
   - Read `q` query param: `r.URL.Query().Get("q")`
   - Lowercase and trim the prefix: `strings.ToLower(strings.TrimSpace(q))`
   - Call `store.ListTags(r.Context(), prefix)`
   - `writeJSON(w, http.StatusOK, tags)`
