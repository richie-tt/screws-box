**9a. DOM references and state variables:**
```javascript
var searchInput = document.getElementById('search-input');
var searchDropdown = document.getElementById('search-dropdown');
var searchListbox = document.getElementById('search-results-listbox');
var dropdownHeader = searchDropdown.querySelector('.search-dropdown-header');
var dropdownEmpty = searchDropdown.querySelector('.search-dropdown-empty');
var dropdownError = searchDropdown.querySelector('.search-dropdown-error');
var clearBtn = document.querySelector('.search-clear-btn');
var spinner = document.querySelector('.search-spinner');

var searchController = null;   // AbortController for in-flight request
var debounceTimer = null;      // setTimeout ID for debounce
var currentResults = [];       // latest search results array
var focusedIndex = -1;         // keyboard-focused result index (-1 = none)
var lastQuery = '';            // last query that was searched
var escapeState = 0;           // tracks layered Escape: 0=in results, 1=dropdown open, 2=text present
```

Guard: `if (!searchInput) return;` (in case template doesn't have search input).

**9b. Helper: highlightMatch(text, query)** (per D-04)
Creates a DocumentFragment with the matching substring wrapped in `<strong>`. Uses `text.toLowerCase().indexOf(query.toLowerCase())` to find the match position. If no match found, returns a plain text node. Uses `document.createTextNode()` and `document.createElement('strong')` -- NO innerHTML.

**9c. Helper: showDropdown() / hideDropdown()**
- `showDropdown()`: Remove `hidden` attribute from searchDropdown, add `visible` class (triggers CSS transition), set `searchInput.setAttribute('aria-expanded', 'true')`.
- `hideDropdown()`: Add `hidden` attribute, remove `visible` class, set `aria-expanded` to `false`, reset `focusedIndex` to -1, clear `aria-activedescendant`.

**9d. Helper: showSpinner() / hideSpinner()**
Toggle the `hidden` attribute on the spinner element.

**9e. Helper: updateClearButton()**
Show clear button (remove `hidden`) if `searchInput.value.length > 0`, hide it otherwise (per D-14).

**9f. Core: renderResults(results, query)** (per D-01, D-02, D-03, D-04, D-05, D-06)

1. Store results in `currentResults`.
2. Clear `searchListbox.innerHTML = ''` (safe -- no user content in the list container itself).
3. If results.length === 0: show empty state (per D-12). Set `dropdownEmpty.textContent` to `"No results for '" + query + "' -- try fewer characters or a different tag"`. Show `dropdownEmpty`, hide `dropdownHeader`, hide `searchListbox`. Call `updateGridHighlights([], null)` to clear highlights. Call `showDropdown()`. Return.
4. Hide `dropdownEmpty`, hide `dropdownError`.
5. Set `dropdownHeader.textContent` to `results.length + (results.length === 1 ? ' result' : ' results')` (per D-05). Show `dropdownHeader`, show `searchListbox`.
6. For each result, create an `<li>` with:
   - `role="option"`, `id="search-result-" + index`, `aria-selected="false"`, `class="search-result"`
   - Inner structure:
     ```
     div.search-result-top
       span.search-result-name  -> highlightMatch(result.name, query)
       span.position-badge      -> result.container_label (textContent)
     div.search-result-tags
       for each tag in result.tags:
         span.tag-chip -> if tag.toLowerCase() === query.toLowerCase(), wrap entire text in <strong>; else use highlightMatch(tag, query)
     ```
   - Click handler on the `<li>`: call `selectResult(index)` (per D-07)
   - Append to `searchListbox`
7. Call `showDropdown()`.
8. Reset `focusedIndex = -1`.
9. Call `updateGridHighlights(results, null)` to highlight matching cells.
10. Auto-scroll grid to first highlighted cell (per D-26): Find the first result's cell via `findCell(results[0].container_id)`, if found call `cell.scrollIntoView({ behavior: 'smooth', block: 'nearest' })`.

**9g. Core: updateGridHighlights(results, focusedContainerId)** (per D-22, D-23, D-25)

1. Build a `matchCounts` object: `{ container_id: count }` from results.
2. Toggle `search-active` class on `.grid-container` based on whether `Object.keys(matchCounts).length > 0`.
3. For each `.grid-cell`:
   - Get `cid = parseInt(cell.dataset.containerId, 10)`
   - Get `count = matchCounts[cid] || 0`
   - `cell.classList.toggle('highlight', count > 0)`
   - `cell.classList.toggle('highlight-focus', cid === focusedContainerId)` (per D-25)
   - Match count badge (per D-22): If `count > 1`, find or create a `span.match-count` child, set `textContent = count`. If `count <= 1`, remove any existing `.match-count` child.
4. If no results, also remove `search-active` class.

**9h. Core: clearSearch()** (per D-14)

1. `searchInput.value = ''`
2. `hideDropdown()`
3. Clear all grid highlights: remove `search-active` from `.grid-container`, remove `highlight` and `highlight-focus` from all `.grid-cell`, remove all `.match-count` badges.
4. `currentResults = []`, `focusedIndex = -1`, `lastQuery = ''`, `escapeState = 0`.
5. `updateClearButton()` (hides the X).
6. If a search AbortController exists, abort it.
7. If `debounceTimer`, clear it.

**9i. Core: performSearch(query)** (per D-08, D-09, D-11)

1. If `searchController`, call `searchController.abort()` (per D-11).
2. Create new `searchController = new AbortController()`.
3. `showSpinner()`.
4. `fetch('/api/search?q=' + encodeURIComponent(query), { signal: searchController.signal })`
5. On success: parse JSON, `hideSpinner()`, set `searchController = null`. Check `searchInput.value.trim()` still matches the query (race condition guard per Pitfall 1). If match, call `renderResults(data.results || [], query)`. If no match, ignore stale response.
6. On error: if `err.name === 'AbortError'`, ignore (expected). Otherwise: `hideSpinner()`, `searchController = null`. Show error state: hide `dropdownHeader`, hide `searchListbox`, set `dropdownError.textContent` to `"Search failed -- check your connection and try again"` (per D-13), show `dropdownError`, call `showDropdown()`. Clear grid highlights.

**9j. Core: selectResult(index)** (per D-07, D-27)

1. Get `result = currentResults[index]`.
2. `hideDropdown()` -- close dropdown but keep search text and highlights (per D-27).
3. Find the target cell via `findCell(result.container_id)`.
4. If cell found: `cell.scrollIntoView({ behavior: 'smooth', block: 'nearest' })`, then call `expandCell(cell)` to open the panel (per D-07).
5. `pulseCell(cell)` for visual feedback.

**9k. Event: searchInput 'input' handler** (per D-08, D-09, D-15)

1. `updateClearButton()`.
2. If `debounceTimer`, `clearTimeout(debounceTimer)`.
3. `var query = searchInput.value.trim()`.
4. If `query.length < 2` (per D-09): call `hideDropdown()`, clear grid highlights (remove `search-active`, `highlight`, `highlight-focus` from all cells, remove `.match-count` badges), `hideSpinner()`, abort any in-flight request. Return.
5. If an item panel is open, close it first (per D-24): `if (expandedPanel) collapseCell();`
6. `showSpinner()`.
7. `debounceTimer = setTimeout(function() { performSearch(query); }, 200)` (per D-08).

**9l. Event: searchInput 'keydown' handler** (per D-16, D-17, D-20, D-21)

Handle these keys:
- **ArrowDown** (per D-16): If dropdown is visible and has results, `e.preventDefault()`, set `focusedIndex = 0` (or increment if already in results), update focus. Call `updateFocusedResult()`.
- **ArrowUp** (per D-16): If `focusedIndex > 0`, decrement. If `focusedIndex === 0`, set to -1 (return focus to input). `e.preventDefault()`. Call `updateFocusedResult()`.
- **Enter** (per D-16): If `focusedIndex >= 0`, `e.preventDefault()`, call `selectResult(focusedIndex)`.
- **Escape** (per D-17 -- layered):
  - If `focusedIndex >= 0` (in results): set `focusedIndex = -1`, focus input, update ARIA. `e.preventDefault(); e.stopPropagation();`
  - Else if dropdown is visible: `hideDropdown()`, clear grid highlights. `e.preventDefault(); e.stopPropagation();`
  - Else if `searchInput.value.length > 0`: `clearSearch()`. `e.preventDefault(); e.stopPropagation();`
- **Home** (discretion): If `focusedIndex >= 0`, set `focusedIndex = 0`, update. `e.preventDefault()`.
- **End** (discretion): If `focusedIndex >= 0`, set `focusedIndex = currentResults.length - 1`, update. `e.preventDefault()`.

**9m. Helper: updateFocusedResult()** (per D-19, D-20, D-25)

1. Update all `<li>` `aria-selected` attributes: set `"true"` on focused, `"false"` on others.
2. Set `searchInput.setAttribute('aria-activedescendant', focusedIndex >= 0 ? 'search-result-' + focusedIndex : '')`.
3. If `focusedIndex >= 0`: scroll the focused `<li>` into view within the dropdown via `li.scrollIntoView({ block: 'nearest' })` (per D-20).
4. Update grid highlight-focus: call `updateGridHighlights(currentResults, focusedIndex >= 0 ? currentResults[focusedIndex].container_id : null)` (per D-25).

**9n. Event: clearBtn 'click' handler** (per D-14)
`e.preventDefault(); e.stopPropagation(); clearSearch(); searchInput.focus();`

**9o. Event: document 'keydown' for / shortcut** (per D-18)
```javascript
document.addEventListener('keydown', function(e) {
  if (e.key === '/' && !e.ctrlKey && !e.metaKey && !e.altKey) {
    var tag = document.activeElement.tagName;
    if (tag === 'INPUT' || tag === 'TEXTAREA' || tag === 'SELECT') return;
    if (document.activeElement.isContentEditable) return;
    e.preventDefault();
    searchInput.focus();
  }
});
```
