**SQL Path 1 -- No tag filters (len(tags)==0, query non-empty) per D-08:**
```sql
SELECT i.id, i.container_id, i.name, i.description,
       c.col, c.row, i.created_at, i.updated_at,
       GROUP_CONCAT(t.name, '|') AS tag_list
FROM item i
JOIN container c ON c.id = i.container_id
LEFT JOIN item_tag it ON it.item_id = i.id
LEFT JOIN tag t ON t.id = it.tag_id
WHERE LOWER(i.name) LIKE ? ESCAPE '\'
   OR t.name = ?
   OR LOWER(COALESCE(i.description, '')) LIKE ? ESCAPE '\'
GROUP BY i.id, i.container_id, i.name, i.description,
         c.col, c.row, i.created_at, i.updated_at
ORDER BY c.col ASC, c.row ASC
LIMIT 51
```
Args: likePattern (e.g. `%bolt%`), query (exact tag match), likePattern again.

**SQL Path 2 -- With tag filters (len(tags)>0) per D-09/D-10:**
```sql
SELECT i.id, i.container_id, i.name, i.description,
       c.col, c.row, i.created_at, i.updated_at,
       GROUP_CONCAT(t.name, '|') AS tag_list
FROM item i
JOIN container c ON c.id = i.container_id
JOIN item_tag it ON it.item_id = i.id
JOIN tag t ON t.id = it.tag_id
WHERE t.name IN (?, ?, ?)
GROUP BY i.id, i.container_id, i.name, i.description,
         c.col, c.row, i.created_at, i.updated_at
HAVING COUNT(DISTINCT t.name) = 3
ORDER BY c.col ASC, c.row ASC
LIMIT 51
```
When query is also non-empty, append after HAVING:
```sql
AND (LOWER(i.name) LIKE ? ESCAPE '\' OR LOWER(COALESCE(i.description, '')) LIKE ? ESCAPE '\')
```
Note: text filter goes in HAVING (not WHERE) to avoid filtering before GROUP BY count (Pitfall 2 from RESEARCH.md).

**Compute matched_on server-side (per D-22):**
```go
func computeMatchedOn(name string, description *string, tags []string, query string, tagsActive bool) []string {
    if query == "" {
        return nil
    }
    var matched []string
    lq := strings.ToLower(query)
    if strings.Contains(strings.ToLower(name), lq) {
        matched = append(matched, "name")
    }
    if description != nil && strings.Contains(strings.ToLower(*description), lq) {
        matched = append(matched, "description")
    }
    if !tagsActive {
        for _, tag := range tags {
            if tag == lq {
                matched = append(matched, "tag")
                break
            }
        }
    }
    return matched
}
```

**Build container label:** Format as `"{col}{rowLetter}"` using existing pattern: col is 1-based integer, row is uppercase letter (A=1, B=2, ...). Use `string(rune('A' + row - 1))` and `strconv.Itoa(col)`.

**LIMIT 51 trick:** Fetch 51 rows. If 51 returned, total_count = actual count via separate `COUNT(*)` query (or just set total_count > 50 and truncate results to 50). Simpler approach: always run a parallel count query without LIMIT when results hit 51. Or: fetch all with LIMIT 51, if len == 51, run count query; else total_count = len.

For the count query, reuse same WHERE/HAVING but wrap in `SELECT COUNT(*) FROM (... without LIMIT)`.

**Return value:** `&model.SearchResponse{Results: results, TotalCount: totalCount}`.

When query=="" and len(tags)==0, return `&model.SearchResponse{Results: []model.SearchResult{}, TotalCount: 0}`.
