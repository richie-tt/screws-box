```go
func (m *MemoryStore) DeleteByAuthMethod(_ context.Context, method string) (int, error) {
    m.mu.Lock()
    defer m.mu.Unlock()
    count := 0
    for id, sess := range m.sessions {
        if sess.AuthMethod == method {
            delete(m.sessions, id)
            count++
        }
    }
    return count, nil
}
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-14-01 | Information Disclosure | GetOIDCConfig | mitigate | GetOIDCConfigMasked strips ClientSecret before API response; handlers MUST use masked version for JSON responses |
| T-14-02 | Information Disclosure | encryption_key in shelf table | accept | Single-user home app; DB file access implies full app access. Encryption key protects in-transit cookies, not at-rest data |
