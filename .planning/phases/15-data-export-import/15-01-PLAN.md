    **internal/store/export.go** -- Two methods on *Store:

    `ExportAllData(ctx context.Context) (*model.ExportData, error)`:
    1. Query shelf: `SELECT id, name, rows, cols FROM shelf LIMIT 1`
    2. Query all containers: `SELECT id, col, row FROM container WHERE shelf_id = ? ORDER BY row, col`
    3. For each container, query items: `SELECT id, name, description FROM item WHERE container_id = ? ORDER BY name`
    4. For each item, query tags: `SELECT t.name FROM tag t JOIN item_tag it ON it.tag_id = t.id WHERE it.item_id = ? ORDER BY t.name`
    5. Build nested tree. Set version=1, exported_at=time.Now().UTC().Format(time.RFC3339)
    6. Container labels via model.LabelFor(col, row)
    7. Ensure Items is never nil (use empty slice `[]ExportItem{}` if no items)
    8. Ensure Tags is never nil (use empty slice `[]string{}` if no tags)

    Optimization: batch items and tags with 2 queries total instead of per-container:
    - `SELECT id, container_id, name, description FROM item ORDER BY container_id, name`
    - `SELECT it.item_id, t.name FROM item_tag it JOIN tag t ON t.id = it.tag_id ORDER BY it.item_id, t.name`
    Then group in Go by container_id and item_id.

    `ImportAllData(ctx context.Context, data *model.ExportData) error`:
    1. Validate version: if data.Version != 1, return `fmt.Errorf("unsupported version %d", data.Version)`
    2. Begin transaction: `s.conn.BeginTx(ctx, nil)` + `defer deferRollback(tx)`
    3. Get shelf ID: `SELECT id FROM shelf LIMIT 1`
    4. Delete in FK order: `DELETE FROM item_tag`, `DELETE FROM item`, `DELETE FROM tag`, `DELETE FROM container WHERE shelf_id = ?`
    5. Update shelf: `UPDATE shelf SET name = ?, rows = ?, cols = ? WHERE id = ?` (per D-06 auto-resize)
    6. Insert containers: `INSERT INTO container (shelf_id, col, row) VALUES (?, ?, ?)` for each container in data.Shelf.Containers
    7. Insert items per container: get container ID by position `SELECT id FROM container WHERE shelf_id = ? AND col = ? AND row = ?`, then `INSERT INTO item (container_id, name, description) VALUES (?, ?, ?)`
    8. Insert tags with dedup: `INSERT OR IGNORE INTO tag (name) VALUES (?)`, then get tag ID: `SELECT id FROM tag WHERE name = ?`, then link: `INSERT INTO item_tag (item_id, tag_id) VALUES (?, ?)`
    9. `tx.Commit()`

    **internal/server/handler_export.go** -- New file with export handler per D-14, D-17:

    ```go
    package server

    import (
        "encoding/json"
        "fmt"
        "log/slog"
        "net/http"
        "time"
    )

    func (srv *Server) handleExport() http.HandlerFunc {
        return func(w http.ResponseWriter, r *http.Request) {
            data, err := srv.store.ExportAllData(r.Context())
            if err != nil {
                slog.Error("export failed", "err", err)
                http.Error(w, "export failed", http.StatusInternalServerError)
                return
            }
            filename := fmt.Sprintf("screws-box-export-%s.json", time.Now().Format("2006-01-02"))
            w.Header().Set("Content-Type", "application/json")
            w.Header().Set("Content-Disposition", fmt.Sprintf(`attachment; filename=%q`, filename))
            enc := json.NewEncoder(w)
            enc.SetIndent("", "  ")
            if err := enc.Encode(data); err != nil {
                slog.Error("export encode failed", "err", err)
            }
        }
    }
    ```
