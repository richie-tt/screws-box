   - Get current session via `srv.sessions.GetSession(r)` to identify admin's session ID.
   - Call `srv.sessions.ListSessions(r.Context())`.
   - Map each `*session.Session` to `SessionInfo`:
     - `CreatedAt` and `LastActivity`: format as `"2006-01-02 15:04"` (local time).
     - `ExpiresIn`: calculate `srv.sessions.TTL() - time.Since(sess.LastActivity)`, format as relative string: if >1h use `"Xh"`, if >1m use `"Xm"`, else `"<1m"`. If expired, use `"expired"`.
     - `IsCurrent`: compare `sess.ID` with current session's ID.
   - Sort by: current session first, then by LastActivity descending.
   - Return JSON array with `writeJSON(w, http.StatusOK, sessions)`.

   **handleRevokeSession()**: DELETE /api/sessions/{sessionID} handler.
   - Extract `sessionID` from URL path param: `chi.URLParam(r, "sessionID")`.
   - Validate format: must be 64 hex chars (256-bit session ID). Return 400 if invalid.
   - Get current session ID from cookie. Compare with target. If same, return 403 JSON `{"error": "cannot revoke your own session"}` per D-18.
   - Call `srv.sessions.DeleteSession(r.Context(), sessionID)`.
   - Return 200 JSON `{"ok": true}`.

   **handleRevokeAllOthers()**: DELETE /api/sessions handler.
   - Get current session ID from cookie.
   - Call `srv.sessions.ListSessions(r.Context())`.
   - For each session where ID != current session ID, call `srv.sessions.DeleteSession`.
   - Count deleted. Return 200 JSON `{"ok": true, "count": N}`.

   Add a helper for relative time formatting:
   ```go
   func formatExpiresIn(lastActivity time.Time, ttl time.Duration) string {
       remaining := ttl - time.Since(lastActivity)
       if remaining <= 0 {
           return "expired"
       }
       if remaining >= time.Hour {
           return fmt.Sprintf("%dh", int(remaining.Hours()))
       }
       if remaining >= time.Minute {
           return fmt.Sprintf("%dm", int(remaining.Minutes()))
       }
       return "<1m"
   }
   ```

       } else if (!e.shiftKey && document.activeElement === last) {
